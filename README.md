# mapbot
Karten Software des BKHs.
<br />
Wortspiel aus "Robot" und "Map", gleichzeitige Anspielung auf Mapbot.

## Controls
|Key|Funktion|
|-|-|
|W, A, S, D, <br/> &uarr;, &larr;, &darr;, &rarr;|Bewegung|
|CTRL, Shift|Sprint|
|LMB|Zielort bestimmen|
|RMB|Route löschen|
|Q, E|Drehung der Kamera|
|Mausrad|Rein- und Rauszoomen|
|C|Zwischen Kamera Modi wechseln|
|R|Reset|
